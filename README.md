# Self-Guided Cultural Tour

## Project Overview
This project is a self-guided cultural tour that allows users to explore various artworks and their associated information through a web interface. The tour includes audio playback, text descriptions, and navigation between different artworks.

## Features
- **Responsive Design**: The application adapts to various screen sizes for optimal viewing on desktops and mobile devices.
- **Audio Player**: Users can listen to audio descriptions of each artwork, with play, pause, and seek functionality.
- **Dynamic Content Loading**: Text descriptions are loaded dynamically from external files, providing flexibility and ease of updates.

## Technologies Used
- HTML
- CSS
- JavaScript
- Fetch API for loading external text files

## Usage
- Navigate through different artworks using the previous and next buttons.
- Click the play button to listen to audio descriptions.
- Seek through the audio using the seek bar.
